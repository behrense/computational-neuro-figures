# computational-neuro-figures

A collection of figures, visualizations, and simulations created while studying computational neuroscience.

## Overview

This repository contains reproducible figures generated from Python code, covering concepts in computational neuroscience, neural dynamics, and data visualization.

## Topics

- Neuron models
- Spike trains
- Population activity
- Neural dynamics
- Probability and statistics
- Signal processing
- Scientific visualization

## Tools

- Python
- NumPy
- Matplotlib
- SciPy
- Jupyter Notebook

## Repository Structure

figures/        Exported images
notebooks/      Interactive notebooks
scripts/        Figure generation scripts
assets/         Supporting files

## Goal

Build a growing collection of computational neuroscience figures while improving scientific programming and visualization skills.
